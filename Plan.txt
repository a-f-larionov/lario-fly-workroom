		Описание документа
	
	В этом документ описан план создания полноценной гипергазул игры в жанре "Муха".
	
	
		Этап 1 Клиент Базовая часть

		Задача 1+	Пустое приложение
	Создаем приложение, и выливаем его на телефон, создаем репозиторий на гитхабе

		Задача 2+	Картинка мухи
	Видим муху
		
		Задача 3	Таймер картинки
	Муха по таймеру
	
		Задача 4	Выводим очки
	Видим очки
		
		Задача 5	Обрабатываем нажатие на муху
	Муха исчезает по нажатию, таймер сбрасывается
	
		Задача 6	Начилсние очков
	По нажатию, начислюятся очки
	
		Задача 7	Сброс таймера
	Муха появляется снова, но таймер уменьшается
	
		Задача 8	Проигрыш
	Выводим сообщение об проигрывании
		

		Этап 2 Сервер Базовая часть
		
		Задача 1	Сервер
	Создаем приложение, и выкалыдываем на хостинг, создаем репозиторий на гитхабе
	
		Задача 2	/login
	Авторизация пользователя по его UID?
	
		Задача 3	БД
	Храним данные о пользователе
	
		
		Этап 3. Персистивность
		
		Задача 1. Храним очки пользователя(рекорд)
		
		Задача 2. Храним очкви всех пользователей, последние 8 игр
		
		Задача 3. Клиент загружает свой хайскор
		
		Этап 4. Клиент с меню
		
		Задача 1. Главное меню: Играть, Хайскор
		
		Задача 2. Кнопка начать заново игру и её обработка
		
		Задача 3. Кнопк в меню и её обработка
		
		Задача 4. Обработк Хайскор: показать хайскор игрока
		
		Задача 5. Показать глобальный хайскор
		
		
		Этап 4. Звуки
		
		Задача 1. звук запуска
		
		Задача 2. звук появления
		
		Задача 3. звук исчезновения
		
		Задача 4. звук проигрыша
		
		Задача 5. звук хайскора
		
		Задача 6. звук кнопок нажатия
